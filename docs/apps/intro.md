# UNN Intro Tool (`unn-intro`)

The **UNN Intro Tool** is the storytelling and tutorial engine for the network. It provides a guided introduction to the UNN lore and technical basics.

### Role & Responsibilities
- **Onboarding**: Introduces new users to the concept of the "Underground Node Network."
- **Lore Delivery**: Delivers a narrative-driven experience about the decentralized underworld.
- **Tutorial**: Explains basic commands and how navigation works before users jack in.

### Key Logic
- Uses a sequenced terminal interface to present text and "hacking" simulations.
- Acts as the first point of contact for many users discovering the network.
