# UNN Applications

The Underground Node Network consists of three core applications that work together to create the decentralized environment.

### [Entrypoint](entrypoint.md) (`unn-entrypoint`)
The signaling and discovery hub.
- **Role**: Coordinates hole-punching and user registration.
- **Vibe**: The landing pad of the network.

### [Room Node](room.md) (`unn-room`)
The user-hosting environment.
- **Role**: Hosts personal rooms, chat, and "Doors" (interactive programs).
- **Vibe**: The personal home/node in the underground.

### [SSH Wrapper](ssh-wrapper.md) (`unn-ssh`)
The automated navigation tool.
- **Role**: High-speed teleportation and download management.
- **Vibe**: The professional's tool for traversing the mesh.

### Other Tools
